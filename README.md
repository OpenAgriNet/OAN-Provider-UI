# OAN Provider

The OAN Provider app is designed for content creators who wish to manage and publish content to the OAN Seeker app. The provider app allows you to easily create, view, edit, and delete content, including titles, descriptions, and media associated with the content. 

## Features

- **Publish Content:** Add new content with titles, descriptions, and media files (images, videos, etc.).
- **List Content:** View all the content you've created, and edit or delete them as needed.

## Installation

To get started, clone the repository and install the required dependencies:

```bash
git clone https://github.com/tekdi/OAN-Provider-UI
```

Navigate to the project folder:

```bash
cd OAN-Provider-UI
```

Install dependencies:

```bash
npm install
```

Start the development server with the following command:

```bash
npm run dev
```

## Usage

### Home Page

The home page has two main tabs:

- **Publish Content:** Use this tab to create new content by providing a title, description, and adding media files.
- **List Content:** View the list of content you have created, with options to edit or delete them.

### Adding Content

1. Navigate to the **Publish Content** tab.
2. Enter the title and description for the content.
3. Upload any media (images, videos, etc.).
4. Click **Publish** to save your content.

### Managing Content

1. Go to the **List Content** tab.
2. Here, you can see all your created content.
3. To edit or delete a content item, simply click on the corresponding action button next to it.
